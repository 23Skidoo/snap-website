Snap Framework

A Web Framework for Haskell

What is Snap Framework?

Snap Framework is a simple web development framework similar to Django or Ruby on Rails but with the power of the purely functional language, Haskell behind it. Snap stands on the shoulders of another exciting Haskell project known as Happstack. While Happstack is a great project with some interesting goals I found it lacks many features web developers expect from a web framework. Snap brings together Happstack with features such as templates and relational database interfaces so that real world web developers who want to use Haskell can get the job done faster.

Why Haskell?

Haskell is a full featured language with a powerful compiler (GHC) behind it and possibly even more important one of the best communities of any language in use today.

Many people see Haskell as a difficult language to understand and program in but this reputation is simply not accurate. When I first started developing my own site in Haskell it was purely for the novelty of trying a new language out. After some time though, not only did I enjoy doing web development in Haskell I found it to be monumentally easier then it would have been in a conventional language like Python or Perl.

I don't know Haskell and I fear it.

No problem we all start out young and afraid, I highly reccomend starting out with Learn You a Haskell for Great Good then following up with some Real World Haskell. They are both free and very readable, so you have no excuse not to learn Haskell.

What goals does Snap have?

Snap has the following goals in mind:

  • Rapid Development.
  • Safer Development.
  • Development that lends itself toward collaboration with non (Haskell) programmers.
  • Sites that are scalable.
  • Site that are easy and enjoyable to maintain.

Where is the project at?

Snap is still in the earliest stages of development. Much is still being planned and the foundations of the project are still very much in the works. Release the source code at this early point would likely be counter productive but I expect to have a first release of the code within the coming months. Snap will of course be an open source project in the future, so I hope anyone interested in lending a hand in development will keep this site bookmarked.
