What is the Snap Framework?

Snap is a simple web development framework in development, written in the purely functional programming language Haskell.

Snap aims to be the de facto web toolkit for Haskell, on the basis of:

  • High performance
  • High design standards
  • Simplicity and ease of use, even for Haskell beginners
  • Excellent documentation
  • Robustness and high test coverage

What is the project's status?

Snap is still in the planning and design stages.
